Description

The idea for this project began when I took a Udemy course to learn how to build a statistical arbitrage trading bot. During the course, when it came time to set the parameters for trading signals—specifically the z-score threshold and the window for the rolling mean used to calculate z-scores—the instructor provided some numbers that had worked for him in backtesting but did not explain how those values were determined. This piqued my interest, as I had recently learned about using Monte Carlo simulations to optimize asset weighting in a stock portfolio. I realized that Monte Carlo simulation could be a powerful technique to explore the parameter space in this context.

Monte Carlo simulations allow for a broad exploration of the parameter space, where there might not be just one optimal solution but potentially several regions of near-optimal parameterizations. This contrasts with a greedy algorithm, which may get stuck in a local maximum or minimum, potentially missing a global optimum or other viable solutions.

Through my research, I’ve found that a single-asset mean reversion strategy tends to be more profitable than a multi-asset statistical arbitrage strategy. However, this approach introduces new challenges related to market neutrality, which must be addressed. I'm investigating a two-step approach: applying PCA to the assets' returns covariance matrix to extract key features, followed by k-means clustering to identify distinct asset groups. By pairing assets from the same cluster and adjusting my trading algorithm to be short one asset and long the other, I aim to enhance market neutrality.

While this strategy carries potential risks, I believe it warrants further analysis and exploration to determine its viability as a market-neutral approach that can be sustainably profitable.
